Detailed Description

A Parameter representing a modifiable signed 32-bit integer value.

Parameters:
cThe Parameters collection to add this Parameter to.
tagThe permanent Tag for this Parameter . This is used for saving this Parameter to a config file, and therefore should be unique to every other parameter tag.
nameThe permanent Name of this Parameter . This will be displayed when any controls for this Parameter are displayed.
descriptionA permanent description of this Parameter . At the moment, this is not displayed with this Parameter .
minThe permanent minimum signed 32-bit value which this Parameter may represent.
initialThe initial signed 32-bit value which this Parameter represents. If this value is less than min, it will be set to min .
maxThe permanent maximum signed 32-bit value which this Parameter may represent.
snapThe permanent snap resolution of this parameter. If this parameter changes, it is floored to a multiple of this snapping value.
